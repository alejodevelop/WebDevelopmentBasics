Computadora - Resumen

El mundo de hoy es un mundo rico en información y se ha convertido en una necesidad para que todos conozcan las computadoras. Una computadora es un dispositivo electrónico de procesamiento de datos que acepta y almacena la entrada de datos, procesa la entrada de datos y genera la salida en un formato requerido.

El propósito de este tutorial es presentarte las computadoras y sus fundamentos.

Funcionalidades de una computadora

Si lo vemos en un sentido muy amplio, cualquier computadora digital realiza las siguientes cinco funciones:

Paso 1 - Toma los datos como entrada.

Paso 2 : almacena los datos / instrucciones en su memoria y los utiliza según sea necesario.

Paso 3 : procesa los datos y los convierte en información útil.

Paso 4 - Genera la salida.

Paso 5 - Controla todos los cuatro pasos anteriores.

Ventajas de las computadoras

A continuación se presentan ciertas ventajas de las computadoras.

Alta velocidad

  • La computadora es un dispositivo muy rápido.

  • Es capaz de realizar el cálculo de una gran cantidad de datos.

  • La computadora tiene unidades de velocidad en microsegundos, nanosegundos e incluso en picosegundos.

  • Puede realizar millones de cálculos en unos pocos segundos en comparación con el hombre que pasará muchos meses para realizar la misma tarea.

Exactitud

  • Además de ser muy rápidos, las computadoras son muy precisas.

  • Los cálculos son 100% sin errores.

  • Las computadoras realizan todos los trabajos con 100% de precisión, siempre que la entrada sea correcta.

Capacidad de almacenamiento

  • La memoria es una característica muy importante de las computadoras.

  • Una computadora tiene mucha más capacidad de almacenamiento que los seres humanos.

  • Puede almacenar gran cantidad de datos.

  • Puede almacenar cualquier tipo de datos como imágenes, videos, texto, audio, etc.

Diligencia

  • A diferencia de los seres humanos, una computadora está libre de monotonía, cansancio y falta de concentración.

  • Puede funcionar continuamente sin ningún error y aburrimiento.

  • Puede realizar tareas repetidas con la misma velocidad y precisión.

Versatilidad

  • Una computadora es una máquina muy versátil.

  • Una computadora es muy flexible para realizar los trabajos a realizar.

  • Esta máquina se puede utilizar para resolver los problemas relacionados con varios campos.

  • En una instancia, puede estar resolviendo un problema científico complejo y en el momento siguiente puede estar jugando un juego de cartas.

Confiabilidad

  • Una computadora es una máquina confiable.

  • Los componentes electrónicos modernos tienen una larga vida.

  • Las computadoras están diseñadas para facilitar el mantenimiento.

Automatización

  • La computadora es una máquina automática.

  • La automatización es la capacidad de realizar una tarea determinada automáticamente. Una vez que la computadora recibe un programa, es decir, el programa se almacena en la memoria de la computadora, entonces el programa y las instrucciones pueden controlar la ejecución del programa sin interacción humana.

Reducción en papeleo y costo

  • El uso de computadoras para el procesamiento de datos en una organización lleva a una reducción en el papeleo y acelera el proceso.

  • Como los datos en los archivos electrónicos se pueden recuperar cuando sea necesario, el problema del mantenimiento de una gran cantidad de archivos en papel se reduce.

  • Aunque la inversión inicial para instalar una computadora es alta, reduce sustancialmente el costo de cada una de sus transacciones.

Desventajas de las computadoras

A continuación se presentan ciertas desventajas de las computadoras.

No IQ

  • Una computadora es una máquina que no tiene inteligencia para realizar ninguna tarea.

  • Cada instrucción tiene que ser dada a la computadora.

  • Una computadora no puede tomar ninguna decisión por sí misma.

Dependencia

  • Funciona según las instrucciones del usuario, por lo que es totalmente dependiente de los seres humanos.

Ambiente

  • El entorno operativo de la computadora debe estar libre de polvo y ser adecuado.

Sin sentimientos

  • Las computadoras no tienen sentimientos ni emociones.

  • No puede emitir juicios basados ​​en sentimientos, gustos, experiencias y conocimientos a diferencia de los humanos.

Aplicaciones computacionales

En este capítulo, discutiremos la aplicación de computadoras en varios campos.

Negocio

Una computadora tiene una alta velocidad de cálculo, diligencia, precisión, confiabilidad o versatilidad, lo que la ha convertido en una parte integrada en todas las organizaciones empresariales.

La computadora es usada en organizaciones de negocios para:

  • Cálculos de nómina
  • Presupuesto
  • Análisis de ventas
  • Previsión financiera
  • Gestión de base de datos de empleados
  • Mantenimiento de stocks, etc.

Bancario

Hoy en día, la banca es casi totalmente dependiente de las computadoras.

Los bancos ofrecen las siguientes facilidades:

  • Facilidad de contabilidad en línea, que incluye verificar el saldo actual, hacer depósitos y sobregiros, verificar cargos por intereses, acciones y registros de fideicomisarios.

  • Los cajeros automáticos que están completamente automatizados hacen que sea más fácil para los clientes tratar con los bancos.

Seguro

Las compañías de seguros mantienen todos los registros actualizados con la ayuda de las computadoras. Las compañías de seguros, casas financieras y firmas de corretaje de valores están usando computadoras en gran medida para sus preocupaciones.

Las compañías de seguros mantienen una base de datos de todos los clientes con información que muestra:

  • Procedimiento para continuar con las políticas.
  • Fecha de inicio de las pólizas.
  • Siguiente pago de una póliza
  • Fecha de vencimiento
  • Intereses vencidos
  • Beneficios de supervivencia
  • Prima

Educación

La computadora ayuda a proporcionar muchas instalaciones en el sistema educativo.

  • La computadora proporciona una herramienta en el sistema educativo conocido como CBE (Computer Based Education).

  • CBE implica control, entrega y evaluación de aprendizaje.

  • La educación en computación está aumentando rápidamente la gráfica de la cantidad de estudiantes de computación.

  • Hay una serie de métodos en los que las instituciones educativas pueden usar una computadora para educar a los estudiantes.

  • Se utiliza para preparar una base de datos sobre el rendimiento de un estudiante y el análisis se lleva a cabo sobre esta base.

Márketing

En marketing, los usos de la computadora son los siguientes:

  • Publicidad : con las computadoras, los profesionales de la publicidad crean arte y gráficos, escriben y revisan copias, e imprimen y difunden anuncios con el objetivo de vender más productos.

  • Compras en el hogar: la compra en el hogar ha sido posible mediante el uso de catálogos computarizados que brindan acceso a la información del producto y permiten que los clientes completen el ingreso directo de pedidos.

Cuidado de la salud

Las computadoras se han convertido en una parte importante en hospitales, laboratorios y dispensarios. Se están utilizando en hospitales para mantener el registro de pacientes y medicamentos. También se utiliza en el escaneo y diagnóstico de diferentes enfermedades. ECG, EEG, ecografías y tomografías computarizadas, etc. también se realizan mediante máquinas computarizadas.

Los siguientes son algunos de los principales campos de la atención médica en los que se utilizan las computadoras

  • Sistema de diagnóstico : las computadoras se utilizan para recopilar datos e identificar la causa de la enfermedad.

  • Sistema de diagnóstico de laboratorio : se pueden realizar todas las pruebas y los informes se preparan por computadora.

  • Sistema de monitorización del paciente : se usa para verificar si hay signos de anomalías en el paciente, como en el caso de un paro cardíaco, un ECG, etc.

  • Sistema de información farmacéutica : la computadora se usa para verificar las etiquetas de los medicamentos, las fechas de caducidad, los efectos secundarios dañinos, etc.

  • Cirugía - Hoy en día, las computadoras también se utilizan para realizar cirugía.

Diseño de ingeniería

Las computadoras son ampliamente utilizadas para fines de ingeniería.

Una de las áreas principales es CAD (diseño asistido por computadora) que proporciona creación y modificación de imágenes. Algunos de los campos son:

  • Ingeniería estructural : requiere análisis de tensión y deformación para el diseño de barcos, edificios, presupuestos, aviones, etc.

  • Ingeniería industrial : las computadoras se ocupan del diseño, la implementación y la mejora de sistemas integrados de personas, materiales y equipos.

  • Ingeniería arquitectónica : las computadoras ayudan a planificar ciudades, a diseñar edificios, a determinar una variedad de edificios en un sitio utilizando dibujos en 2D y 3D.

Militar

Las computadoras se utilizan en gran medida en la defensa. Los tanques modernos, misiles, armas, etc. Los militares también emplean sistemas de control computarizados. Algunas áreas militares donde se ha utilizado una computadora son:

  • Control de misiles
  • Comunicacion militar
  • Operación militar y planificación
  • Armas inteligentes

Comunicación

La comunicación es una forma de transmitir un mensaje, una idea, una imagen o un discurso que la persona a quien se refiere entiende y entiende clara y correctamente. Algunas áreas principales en esta categoría son -

  • Email
  • Chateando
  • Usenet
  • FTP
  • Telnet
  • Videoconferencia

Gobierno

Las computadoras juegan un papel importante en los servicios gubernamentales. Algunos campos importantes en esta categoría son:

  • Presupuestos
  • Departamento de impuestos de ventas
  • Departamento de impuestos
  • Cálculo de la relación hombre / mujer.
  • Informatización de las listas de votantes.

  • Predicción del tiempo

Inicialmente, el término generación se usó para distinguir entre diferentes tecnologías de hardware. Hoy en día, la generación incluye tanto hardware como software, que en conjunto conforman un sistema informático completo.

Hay cinco generaciones de computadoras conocidas hasta la fecha. Cada generación ha sido discutida en detalle junto con su período de tiempo y características. En la siguiente tabla, se han mencionado las fechas aproximadas de cada generación, que normalmente se aceptan.

Computadora - Tipos

Las computadoras pueden clasificarse ampliamente por su velocidad y potencia de cálculo.

S.No. Tipo Presupuesto
1 PC (computadora personal) Es un sistema informático de un solo usuario que tiene un microprocesador moderadamente potente.
2 Puesto de trabajo También es un sistema informático de un solo usuario, similar a una computadora personal, sin embargo, tiene un microprocesador más potente.
3 Mini computadora Es un sistema informático multiusuario, capaz de soportar a cientos de usuarios simultáneamente.
4 Marco principal Es un sistema informático multiusuario, capaz de soportar a cientos de usuarios simultáneamente. La tecnología de software es diferente de la minicomputadora.
5 Supercomputadora Es una computadora extremadamente rápida, que puede ejecutar cientos de millones de instrucciones por segundo.

PC (computadora personal)

Una PC puede definirse como una computadora pequeña, relativamente barata, diseñada para un usuario individual. Las PC se basan en la tecnología de microprocesador que permite a los fabricantes colocar una CPU completa en un chip. Las empresas utilizan computadoras personales para el procesamiento de textos, contabilidad, publicación de escritorio y para ejecutar aplicaciones de gestión de bases de datos y hojas de cálculo. En el hogar, el uso más popular de las computadoras personales es jugar y navegar por Internet.

Si bien las computadoras personales están diseñadas como sistemas de un solo usuario, estos sistemas normalmente están conectados entre sí para formar una red. En términos de potencia, los modelos actuales de alta gama de Macintosh y PC ofrecen la misma potencia de cómputo y capacidad gráfica que las estaciones de trabajo de gama baja de Sun Microsystems, Hewlett-Packard y Dell.

Puesto de trabajo

Workstation es una computadora que se usa para aplicaciones de ingeniería (CAD / CAM), publicación de escritorio, desarrollo de software y otros tipos de aplicaciones que requieren una cantidad moderada de potencia de cómputo y capacidades gráficas de calidad relativamente alta.

Las estaciones de trabajo generalmente vienen con una gran pantalla gráfica de alta resolución, una gran cantidad de RAM, soporte de red incorporado y una interfaz gráfica de usuario. La mayoría de las estaciones de trabajo también tienen un dispositivo de almacenamiento masivo, como una unidad de disco, pero un tipo especial de estación de trabajo, llamada estación de trabajo sin disco, viene sin una unidad de disco.

Los sistemas operativos comunes para estaciones de trabajo son UNIX y Windows NT. Al igual que las PC, las estaciones de trabajo también son computadoras de un solo usuario, como las PC, pero generalmente están conectadas entre sí para formar una red de área local, aunque también pueden usarse como sistemas independientes.

Miniordenador

Es un sistema de multiprocesamiento de tamaño medio capaz de admitir hasta 250 usuarios simultáneamente.

Marco principal

El mainframe es muy grande en tamaño y es una computadora costosa capaz de soportar cientos o incluso miles de usuarios simultáneamente. Mainframe ejecuta muchos programas al mismo tiempo y soporta muchas ejecuciones simultáneas de programas.

Supercomputadora

Las supercomputadoras son una de las computadoras más rápidas disponibles en la actualidad. Las supercomputadoras son muy caras y se emplean para aplicaciones especializadas que requieren una gran cantidad de cálculos matemáticos (procesamiento de números).

Por ejemplo, pronóstico del tiempo, simulaciones científicas, gráficos (animados), cálculos dinámicos de fluidos, investigación de energía nuclear, diseño electrónico y análisis de datos geológicos (por ejemplo, en prospección petroquímica).

Componentes de la computadora

Todos los tipos de computadoras siguen la misma estructura lógica básica y realizan las siguientes cinco operaciones básicas para convertir datos de entrada sin procesar en información útil para sus usuarios.

S.No. Operación Descripción
1 Toma de entrada El proceso de introducir datos e instrucciones en el sistema informático.
2 Almacenamiento de datos Guardar datos e instrucciones para que estén disponibles para el procesamiento cuando sea necesario.
3 Procesando datos Realizar operaciones aritméticas y lógicas en los datos para convertirlos en información útil.
4 Información de salida El proceso de producir información útil o resultados para el usuario, como un informe impreso o una pantalla visual.
5 Controlar el flujo de trabajo Dirige la manera y la secuencia en que se realizan todas las operaciones anteriores.

Unidad de entrada

Esta unidad contiene dispositivos con la ayuda de los cuales ingresamos datos en la computadora. Esta unidad crea un enlace entre el usuario y la computadora. Los dispositivos de entrada traducen la información en una forma comprensible para la computadora.

CPU (Unidad Central de Procesamiento)

La CPU es considerada como el cerebro de la computadora. CPU realiza todo tipo de operaciones de procesamiento de datos. Almacena datos, resultados intermedios e instrucciones (programa). Controla el funcionamiento de todas las partes de la computadora.

La CPU tiene los siguientes tres componentes:

  • ALU (Unidad de lógica aritmética)
  • Unidad de memoria
  • Unidad de control

Unidad de salida

La unidad de salida consta de dispositivos con la ayuda de los cuales obtenemos la información de la computadora. Esta unidad es un enlace entre la computadora y los usuarios. Los dispositivos de salida traducen la salida de la computadora en una forma comprensible para los usuarios.

Computadora - CPU (Unidad Central de Procesamiento)

La unidad central de procesamiento (CPU) consta de las siguientes características:

  • La CPU es considerada como el cerebro de la computadora.
  • CPU realiza todo tipo de operaciones de procesamiento de datos.
  • Almacena datos, resultados intermedios e instrucciones (programa).
  • Controla el funcionamiento de todas las partes de la computadora.

La propia CPU tiene los siguientes tres componentes.

  • Unidad de memoria o almacenamiento
  • Unidad de control
  • ALU (Unidad de lógica aritmética)

Unidad de memoria o almacenamiento

Esta unidad puede almacenar instrucciones, datos y resultados intermedios. Esta unidad suministra información a otras unidades de la computadora cuando sea necesario. También se conoce como unidad de almacenamiento interno o la memoria principal o el almacenamiento primario o la memoria de acceso aleatorio (RAM).

Su tamaño afecta la velocidad, la potencia y la capacidad. La memoria primaria y la memoria secundaria son dos tipos de memorias en la computadora. Las funciones de la unidad de memoria son:

  • Almacena todos los datos y las instrucciones requeridas para su procesamiento.

  • Almacena resultados intermedios de procesamiento.

  • Almacena los resultados finales del procesamiento antes de que estos resultados se publiquen en un dispositivo de salida.

  • Todas las entradas y salidas se transmiten a través de la memoria principal.

Unidad de control

Esta unidad controla las operaciones de todas las partes de la computadora pero no realiza ninguna operación de procesamiento de datos real.

Las funciones de esta unidad son:

  • Es responsable de controlar la transferencia de datos e instrucciones entre otras unidades de una computadora.

  • Gestiona y coordina todas las unidades del ordenador.

  • Obtiene las instrucciones de la memoria, las interpreta y dirige el funcionamiento de la computadora.

  • Se comunica con los dispositivos de entrada / salida para la transferencia de datos o resultados del almacenamiento.

  • No procesa ni almacena datos.

ALU (Unidad de lógica aritmética)

Esta unidad consta de dos subsecciones a saber,

  • Sección Aritmética
  • Seccion logica

Sección Aritmética

La función de la sección aritmética es realizar operaciones aritméticas como la suma, la resta, la multiplicación y la división. Todas las operaciones complejas se realizan mediante el uso repetitivo de las operaciones anteriores.

Seccion logica

La función de la sección de lógica es realizar operaciones lógicas como la comparación, selección, comparación y combinación de datos.

Teclado

El teclado es el dispositivo de entrada más común y más popular que ayuda a ingresar datos en la computadora. El diseño del teclado es como el de la máquina de escribir tradicional, aunque hay algunas teclas adicionales que se proporcionan para realizar funciones adicionales.

Los teclados son de dos tamaños, 84 teclas o 101/102 teclas, pero ahora los teclados con 104 teclas o 108 teclas también están disponibles para Windows e Internet.

Las teclas en el teclado son las siguientes:

S.No Llaves y Descripción
1

Teclas de escritura

Estas teclas incluyen las teclas de letras (AZ) y las teclas de dígitos (09) que generalmente tienen el mismo diseño que las máquinas de escribir.
2

Teclado numérico

Se utiliza para introducir los datos numéricos o el movimiento del cursor. En general, consiste en un conjunto de 17 teclas que están dispuestas en la misma configuración utilizada por la mayoría de las máquinas sumadoras y calculadoras.
3

Teclas de función

Las doce teclas de función están presentes en el teclado, que están dispuestas en una fila en la parte superior del teclado. Cada tecla de función tiene un significado único y se utiliza para algún propósito específico.
4

Teclas de control

Estas teclas proporcionan control de cursor y pantalla. Incluye cuatro teclas de flecha direccionales. Las teclas de control también incluyen Inicio, Fin, Insertar, Eliminar, Re Pág, Av Pág, Control (Ctrl), Alternativo (Alt), Escape (Esc).
5

Llaves de propósito especial

El teclado también contiene algunas teclas de propósito especial, como Intro, Mayús, Bloqueo de mayúsculas, Bloqueo numérico, Barra espaciadora, Tabulador y Pantalla de impresión.

Ratón

El ratón es el dispositivo señalador más popular. Es un dispositivo muy famoso para el control del cursor que tiene una caja pequeña del tamaño de la palma con una bola redonda en su base, que detecta el movimiento del mouse y envía las señales correspondientes a la CPU cuando se presionan los botones del mouse.

En general, tiene dos botones llamados el botón izquierdo y derecho y una rueda está presente entre los botones. Se puede usar un mouse para controlar la posición del cursor en la pantalla, pero no se puede usar para ingresar texto en la computadora.

Ventajas

  • Fácil de usar
  • No muy caro
  • Mueve el cursor más rápido que las teclas de flecha del teclado.

Palanca de mando

El joystick también es un dispositivo señalador, que se utiliza para mover la posición del cursor en una pantalla de monitor. Es un palo que tiene una bola esférica en sus extremos inferior y superior. La bola esférica inferior se mueve en un casquillo. El joystick se puede mover en las cuatro direcciones.

La función del joystick es similar a la de un ratón. Se utiliza principalmente en el diseño asistido por computadora (CAD) y en juegos de computadora.

Pluma ligera

La pluma de luz es un dispositivo señalador similar a una pluma. Se utiliza para seleccionar un elemento del menú mostrado o dibujar imágenes en la pantalla del monitor. Consiste en una fotocélula y un sistema óptico colocado en un pequeño tubo.

Cuando la punta de un lápiz óptico se mueve sobre la pantalla del monitor y se presiona el botón del lápiz, su elemento sensor de fotocélula detecta la ubicación de la pantalla y envía la señal correspondiente a la CPU.

Bola de seguimiento

La bola de seguimiento es un dispositivo de entrada que se utiliza principalmente en computadoras portátiles o portátiles, en lugar de un mouse. Esta es una bola que está medio insertada y moviendo los dedos sobre la bola, se puede mover el puntero.

Dado que no se mueve todo el dispositivo, una bola de seguimiento requiere menos espacio que un mouse. Una bola de seguimiento viene en varias formas, como una bola, un botón o un cuadrado.

Escáner

El escáner es un dispositivo de entrada, que funciona más como una fotocopiadora. Se utiliza cuando hay cierta información disponible en papel y se transfiere al disco duro de la computadora para su posterior manipulación.

El escáner captura imágenes de la fuente que luego se convierten en un formato digital que se puede almacenar en el disco. Estas imágenes se pueden editar antes de imprimirlas.

Digitalizador

El digitalizador es un dispositivo de entrada que convierte la información analógica en forma digital. El digitalizador puede convertir una señal del televisor o la cámara en una serie de números que podrían almacenarse en una computadora. Pueden ser utilizados por la computadora para crear una imagen de lo que la cámara había apuntado.

El digitalizador también se conoce como Tableta o Tableta gráfica, ya que convierte gráficos y datos pictóricos en entradas binarias. Una tableta gráfica como digitalizador se utiliza para trabajos finos de aplicaciones de dibujo y manipulación de imágenes.

Micrófono

El micrófono es un dispositivo de entrada para ingresar sonido que luego se almacena en forma digital.

El micrófono se usa para varias aplicaciones, como agregar sonido a una presentación multimedia o para mezclar música.

Lector de tarjetas de tinta magnética (MICR)

El dispositivo de entrada MICR se usa generalmente en los bancos ya que hay una gran cantidad de cheques que se procesan todos los días. El número de código y el cheque del banco se imprimen en los cheques con un tipo especial de tinta que contiene partículas de material magnético que se pueden leer en la máquina.

Este proceso de lectura se llama reconocimiento de caracteres de tinta magnética (MICR). Las principales ventajas de MICR es que es rápido y menos propenso a errores.

Lector óptico de caracteres (OCR)

OCR es un dispositivo de entrada que se utiliza para leer un texto impreso.

OCR escanea el texto de forma óptica, carácter por carácter, los convierte en un código legible por máquina y almacena el texto en la memoria del sistema.

Lectores de códigos de barras

El lector de códigos de barras es un dispositivo que se utiliza para leer datos de códigos de barras (datos en forma de líneas claras y oscuras). Los datos de códigos de barras se utilizan generalmente para etiquetar productos, numerar los libros, etc. Puede ser un escáner de mano o puede estar integrado en un escáner estacionario.

El lector de códigos de barras escanea una imagen de código de barras, la convierte en un valor alfanumérico, que luego se envía a la computadora a la que está conectado el lector de códigos de barras.

Lector óptico de marcas (OMR)

OMR es un tipo especial de escáner óptico que se utiliza para reconocer el tipo de marca hecha con un bolígrafo o un lápiz. Se utiliza cuando se selecciona y se marca una de las pocas alternativas.

Se utiliza especialmente para verificar las hojas de respuestas de los exámenes que tienen preguntas de opción múltiple.

Computadora - Dispositivos de salida

Los siguientes son algunos de los dispositivos de salida importantes utilizados en una computadora.

  • Monitores
  • Trazador gráfico
  • Impresora

Monitores

Los monitores, comúnmente llamados como unidad de visualización visual (VDU), son el dispositivo de salida principal de una computadora. Forma imágenes a partir de pequeños puntos, llamados píxeles que se organizan en forma rectangular. La nitidez de la imagen depende de la cantidad de píxeles.

Hay dos tipos de pantallas de visualización utilizadas para monitores.

  • Tubo de rayos catódicos (CRT)
  • Monitor de pantalla plana

Monitor de tubo de rayos catódicos (CRT)

La pantalla CRT está formada por pequeños elementos de imagen llamados píxeles. Cuanto más pequeños sean los píxeles, mejor será la claridad o la resolución de la imagen. Se necesita más de un píxel iluminado para formar un carácter completo, como la letra 'e' en la palabra ayuda.

Se puede mostrar un número finito de caracteres en una pantalla a la vez. La pantalla se puede dividir en una serie de cuadros de caracteres: ubicación fija en la pantalla donde se puede colocar un carácter estándar. La mayoría de las pantallas son capaces de mostrar 80 caracteres de datos horizontalmente y 25 líneas verticalmente.

Hay algunas desventajas de CRT -

  • De gran tamaño
  • Alto consumo de energía

Monitor de pantalla plana

La pantalla plana se refiere a una clase de dispositivos de video que tienen requisitos de volumen, peso y energía reducidos en comparación con el CRT. Puedes colgarlos en las paredes o ponértelos en las muñecas. Los usos actuales de las pantallas planas incluyen calculadoras, videojuegos, monitores, computadora portátil y pantalla gráfica.

La pantalla plana se divide en dos categorías:

  • Pantallas emisivas: las pantallas emisivas son dispositivos que convierten la energía eléctrica en luz. Por ejemplo, panel de plasma y LED (diodos emisores de luz).

  • Pantallas no emisivas: las pantallas no emisivas utilizan efectos ópticos para convertir la luz solar o la luz de alguna otra fuente en patrones gráficos. Por ejemplo, LCD (Liquid-Crystal Device).

Impresoras

La impresora es un dispositivo de salida, que se utiliza para imprimir información en papel.

Hay dos tipos de impresoras:

  • Impresoras de impacto
  • Impresoras sin impacto

Impresoras de impacto

Las impresoras de impacto imprimen los caracteres al pegarlos en la cinta, que luego se presiona en el papel.

Las características de las impresoras de impacto son las siguientes:

  • Costes de consumo muy bajos.
  • Muy ruidoso
  • Útil para impresión a granel debido a su bajo costo
  • Hay contacto físico con el papel para producir una imagen.

Estas impresoras son de dos tipos:

  • Impresoras de caracteres
  • Impresoras de linea

Impresoras de caracteres

Las impresoras de caracteres son las impresoras que imprimen un carácter a la vez.

Estos se dividen en dos tipos:

  • Impresora de matriz de puntos (DMP)
  • Rueda giratoria

Impresora matricial de puntos

En el mercado, una de las impresoras más populares es la impresora matricial. Estas impresoras son populares debido a su facilidad de impresión y precio económico. Cada carácter impreso tiene la forma de un patrón de puntos y la cabeza consta de una Matriz de Pines de tamaño (5 * 7, 7 * 9, 9 * 7 o 9 * 9) que forman un carácter, por lo que es llamada impresora de matriz de puntos.

Ventajas

  • Barato
  • Ampliamente utilizado
  • Se pueden imprimir otros caracteres del idioma.

Desventajas

  • Velocidad lenta
  • Mala calidad

Rueda giratoria

La cabeza está sobre una rueda y los pines correspondientes a los personajes son como pétalos de Daisy (flor), por eso se llama Daisy Wheel Printer. Estas impresoras se utilizan generalmente para el procesamiento de textos en oficinas que requieren que se envíen algunas cartas aquí y allá con una calidad muy buena.

Ventajas

  • Más confiable que DMP
  • Mejor calidad
  • Las fuentes de caracteres se pueden cambiar fácilmente

Desventajas

  • Más lento que DMP
  • Ruidoso
  • Más caro que el DMP

Impresoras de linea

Las impresoras de líneas son las impresoras que imprimen una línea a la vez.

Estos son de dos tipos:

  • Impresora de tambor
  • Impresora de cadena

Impresora de tambor

Esta impresora es como un tambor en forma, por eso se llama impresora de tambor. La superficie del tambor se divide en una serie de pistas. Las pistas totales son iguales al tamaño del papel, es decir, para un ancho de papel de 132 caracteres, el tambor tendrá 132 pistas. Un conjunto de caracteres está grabado en relieve en la pista. Diferentes conjuntos de caracteres disponibles en el mercado son 48 conjuntos de caracteres, 64 y 96 caracteres establecidos. Una rotación del tambor imprime una línea. Las impresoras de tambor son rápidas en velocidad y pueden imprimir de 300 a 2000 líneas por minuto.

Ventajas

  • Muy alta velocidad

Desventajas

  • Muy caro
  • Las fuentes de caracteres no se pueden cambiar

Impresora de cadena

En esta impresora, se utiliza una cadena de juegos de caracteres, por lo que se denomina impresora de cadena. Un conjunto de caracteres estándar puede tener 48, 64 o 96 caracteres.

Ventajas

  • Las fuentes de caracteres se pueden cambiar fácilmente.
  • Se pueden utilizar diferentes idiomas con la misma impresora.

Desventajas

  • Ruidoso

Impresoras sin impacto

Las impresoras sin impacto imprimen los caracteres sin utilizar la cinta. Estas impresoras imprimen una página completa a la vez, por lo que también se llaman Impresoras de páginas.

Estas impresoras son de dos tipos:

  • Impresoras laser
  • Impresoras de inyección de tinta

Características de las impresoras sin impacto

  • Más rápido que las impresoras de impacto
  • No son ruidosos
  • Alta calidad
  • Soporta muchas fuentes y diferentes tamaños de caracteres

Impresoras laser

Estas son impresoras de página sin impacto. Usan luces láser para producir los puntos necesarios para formar los caracteres que se imprimirán en una página.

Ventajas

  • Muy alta velocidad
  • Salida de muy alta calidad.
  • Buena calidad gráfica
  • Soporta muchas fuentes y diferentes tamaños de caracteres

Desventajas

  • Costoso
  • No se puede utilizar para producir copias múltiples de un documento en una sola impresión

Impresoras de inyección de tinta

Las impresoras de inyección de tinta son impresoras de caracteres sin impacto basadas en una tecnología relativamente nueva. Imprimen caracteres rociando pequeñas gotas de tinta sobre papel. Las impresoras de inyección de tinta producen resultados de alta calidad con características presentables.

Hacen menos ruido porque no se realiza ningún martilleo y estos tienen muchos estilos de modos de impresión disponibles. La impresión en color también es posible. Algunos modelos de impresoras de inyección de tinta también pueden producir múltiples copias de impresión.

Ventajas

  • Impresión de alta calidad
  • Más confiable

Desventajas

  • Caro ya que el costo por página es alto
  • Lento en comparación con la impresora láser

Memoria del ordenador

Un recuerdo es como un cerebro humano. Se utiliza para almacenar datos e instrucciones. La memoria de la computadora es el espacio de almacenamiento en la computadora, donde se procesan los datos y se almacenan las instrucciones necesarias para el procesamiento. La memoria se divide en un gran número de pequeñas partes llamadas células. Cada ubicación o celda tiene una dirección única, que varía de cero a tamaño de memoria menos uno. Por ejemplo, si la computadora tiene 64k palabras, entonces esta unidad de memoria tiene 64 * 1024 = 65536 ubicaciones de memoria. La dirección de estas ubicaciones varía de 0 a 65535.

La memoria es principalmente de tres tipos:

  • Memoria caché
  • Memoria primaria / Memoria principal
  • Memoria secundaria

Memoria caché

La memoria caché es una memoria de semiconductores de muy alta velocidad que puede acelerar la CPU. Actúa como un búfer entre la CPU y la memoria principal. Se utiliza para almacenar aquellas partes de datos y programas que la CPU utiliza con mayor frecuencia. Las partes de los datos y programas se transfieren desde el disco a la memoria caché mediante el sistema operativo, desde donde la CPU puede acceder a ellos.

Ventajas

Las ventajas de la memoria caché son las siguientes:

  • La memoria caché es más rápida que la memoria principal.
  • Consume menos tiempo de acceso en comparación con la memoria principal.
  • Almacena el programa que se puede ejecutar en un corto período de tiempo.
  • Almacena datos para uso temporal.

Desventajas

Las desventajas de la memoria caché son las siguientes:

  • La memoria caché tiene una capacidad limitada.
  • Es muy caro.

Memoria primaria (Memoria principal)

La memoria primaria solo contiene aquellos datos e instrucciones en los que la computadora está trabajando actualmente. Tiene una capacidad limitada y los datos se pierden cuando se apaga la alimentación. Generalmente se compone de dispositivo semiconductor. Estos recuerdos no son tan rápidos como los registros. Los datos e instrucciones requeridos para ser procesados ​​residen en la memoria principal. Se divide en dos subcategorías RAM y ROM.

Características de la memoria principal

  • Estas son las memorias de los semiconductores.
  • Es conocido como la memoria principal.
  • Normalmente la memoria es volátil.
  • Los datos se pierden en caso de que se desconecte la alimentación.
  • Es la memoria de trabajo de la computadora.
  • Más rápido que los recuerdos secundarios.
  • Una computadora no puede funcionar sin la memoria primaria.

Memoria secundaria

Este tipo de memoria también se conoce como memoria externa o no volátil. Es más lento que la memoria principal. Estos se utilizan para almacenar datos / información de forma permanente. La CPU no accede directamente a estas memorias, en su lugar, se accede a ellas mediante rutinas de entrada-salida. El contenido de las memorias secundarias se transfiere primero a la memoria principal, y luego la CPU puede acceder a ella. Por ejemplo, disco, CD-ROM, DVD, etc.

Características de la memoria secundaria

  • Estas son memorias magnéticas y ópticas.
  • Es conocida como la memoria de respaldo.
  • Es una memoria no volátil.
  • Los datos se almacenan permanentemente, incluso si la alimentación está apagada.
  • Se utiliza para el almacenamiento de datos en una computadora.
  • La computadora puede funcionar sin la memoria secundaria.
  • Más lento que los recuerdos primarios.

Memoria de acceso aleatorio

RAM (Memoria de acceso aleatorio) es la memoria interna de la CPU para almacenar datos, programas y resultados de programas. Es una memoria de lectura / escritura que almacena datos hasta que la máquina está funcionando. En cuanto se apaga la máquina, se borran los datos.

El tiempo de acceso en la RAM es independiente de la dirección, es decir, cada ubicación de almacenamiento dentro de la memoria es tan fácil de alcanzar como otras ubicaciones y toma la misma cantidad de tiempo. Se puede acceder a los datos en la RAM de forma aleatoria pero es muy costoso.

La memoria RAM es volátil, es decir, los datos almacenados en ella se pierden cuando apagamos la computadora o si hay un fallo de alimentación. Por lo tanto, un sistema de energía ininterrumpible de respaldo (UPS) se usa a menudo con las computadoras. La memoria RAM es pequeña, tanto en términos de su tamaño físico como en la cantidad de datos que puede contener.

RAM es de dos tipos:

  • RAM estática (SRAM)
  • RAM dinámica (DRAM)

RAM estática (SRAM)

La palabra estática indica que la memoria conserva su contenido mientras se suministra energía. Sin embargo, los datos se pierden cuando la alimentación se reduce debido a la naturaleza volátil. Los chips SRAM utilizan una matriz de 6 transistores y no tienen capacitores. Los transistores no requieren energía para evitar fugas, por lo que no es necesario actualizar SRAM de forma regular.

Hay espacio adicional en la matriz, por lo que SRAM usa más chips que DRAM por la misma cantidad de espacio de almacenamiento, lo que hace que los costos de fabricación sean más altos. SRAM se utiliza así como memoria caché y tiene un acceso muy rápido.

Característica de RAM estática

  • Larga vida
  • No es necesario actualizar
  • Más rápido
  • Utilizado como memoria caché
  • Talla grande
  • Costoso
  • Alto consumo de energía

RAM dinámica (DRAM)

DRAM, a diferencia de SRAM, debe actualizarse continuamente para mantener los datos. Esto se hace colocando la memoria en un circuito de actualización que reescribe los datos varios cientos de veces por segundo. La DRAM se usa para la mayoría de las memorias del sistema, ya que es barata y pequeña. Todas las DRAM se componen de celdas de memoria, que se componen de un capacitor y un transistor.

Características de la RAM dinámica

  • Vida útil corta de los datos
  • Necesita ser renovado continuamente
  • Más lento en comparación con SRAM
  • Utilizado como memoria RAM
  • Más pequeño en tamaño
  • Menos costoso
  • Menos consumo de energía

Computadora - Memoria de sólo lectura

ROM significa memoria de solo lectura . La memoria de la cual solo podemos leer pero no podemos escribir en ella. Este tipo de memoria no es volátil. La información se almacena permanentemente en tales memorias durante la fabricación. Una ROM almacena las instrucciones necesarias para iniciar una computadora. Esta operación se conoce como bootstrap . Los chips ROM no solo se usan en la computadora sino también en otros artículos electrónicos como la lavadora y el horno de microondas.

Discutamos ahora los diversos tipos de ROM y sus características.

MROM (ROM enmascarada)

Las primeras ROM fueron dispositivos con cableado que contenían un conjunto de datos o instrucciones preprogramados. Este tipo de ROM se conocen como ROM enmascaradas, que son baratas.

PROM (Memoria de solo lectura programable)

PROM es una memoria de solo lectura que solo un usuario puede modificar una vez. El usuario compra una PROM en blanco e ingresa los contenidos deseados utilizando un programa PROM. Dentro del chip PROM, hay pequeños fusibles que se queman durante la programación. Se puede programar solo una vez y no se puede borrar.

EPROM (Memoria de solo lectura borrable y programable)

La EPROM se puede borrar exponiéndola a la luz ultravioleta durante un tiempo de hasta 40 minutos. Por lo general, un borrador de EPROM logra esta función. Durante la programación, una carga eléctrica queda atrapada en una región de puerta aislada. La carga se retiene por más de 10 años porque la carga no tiene una vía de fuga. Para borrar esta carga, la luz ultravioleta pasa a través de una ventana de cristal de cuarzo (tapa). Esta exposición a la luz ultravioleta disipa la carga. Durante el uso normal, la tapa de cuarzo se sella con un adhesivo.

EEPROM (memoria de solo lectura y borrable eléctricamente programable)

La EEPROM está programada y borrada eléctricamente. Se puede borrar y reprogramar unas diez mil veces. Tanto el borrado como la programación toman alrededor de 4 a 10 ms (milisegundos). En EEPROM, cualquier ubicación puede ser borrada y programada selectivamente. Las EEPROM pueden borrarse un byte a la vez, en lugar de borrar todo el chip. Por lo tanto, el proceso de reprogramación es flexible pero lento.

Ventajas de la ROM

Las ventajas de la ROM son las siguientes:

  • No volátil en la naturaleza
  • No puede ser cambiado accidentalmente
  • Más barato que las memorias RAM
  • Fácil de probar
  • Más confiable que las RAMs
  • Estático y no requiere refrescante.
  • Los contenidos son siempre conocidos y pueden ser verificados.

Ordenador - placa base

La placa base sirve como una plataforma única para conectar todas las partes de una computadora. Conecta la CPU, la memoria, los discos duros, las unidades ópticas, la tarjeta de video, la tarjeta de sonido y otros puertos y tarjetas de expansión directamente o mediante cables. Puede ser considerado como la columna vertebral de una computadora.

Características de la placa base

Una placa base viene con las siguientes características:

  • La placa base varía mucho en el soporte de varios tipos de componentes.

  • La placa base admite un solo tipo de CPU y algunos tipos de memorias.

  • Las tarjetas de video, discos duros, tarjetas de sonido deben ser compatibles con la placa base para funcionar correctamente.

  • Las placas madre, los estuches y las fuentes de alimentación deben ser compatibles para que funcionen correctamente juntos.

Fabricantes Populares

Los siguientes son los fabricantes populares de la placa base.

  • Intel
  • ASUS
  • AOpen
  • UN POCO
  • Biostar
  • Gigabyte
  • MSI

Descripción de la placa base

La placa base está montada dentro de la caja y se sujeta de manera segura a través de pequeños tornillos a través de orificios previamente perforados. La placa base contiene puertos para conectar todos los componentes internos. Proporciona un solo zócalo para la CPU, mientras que para la memoria, normalmente hay una o más ranuras disponibles. Las placas madre proporcionan puertos para conectar la unidad de disquete, el disco duro y las unidades ópticas a través de cables de cinta. La placa base lleva ventiladores y un puerto especial diseñado para suministro de energía.

Hay una ranura para tarjeta periférica frente a la placa base que permite conectar tarjetas de video, tarjetas de sonido y otras tarjetas de expansión a la placa base.

En el lado izquierdo, las placas base llevan una serie de puertos para conectar el monitor, la impresora, el mouse, el teclado, el altavoz y los cables de red. Las placas madre también proporcionan puertos USB, que permiten que los dispositivos compatibles se conecten en forma de plug-in / plug-out. Por ejemplo, pen drive, cámaras digitales, etc.

Ordenador - Puertos

Un puerto es un punto de acoplamiento físico mediante el cual se puede conectar un dispositivo externo a la computadora. También puede ser un punto de acoplamiento programático a través del cual la información fluye desde un programa a la computadora o por Internet.

Características de los puertos.

Un puerto tiene las siguientes características:

  • Los dispositivos externos están conectados a una computadora mediante cables y puertos.

  • Los puertos son ranuras en la placa base en la que se conecta un cable de dispositivo externo.

  • Ejemplos de dispositivos externos conectados a través de puertos son el mouse, el teclado, el monitor, el micrófono, los altavoces, etc.

Discutamos ahora algunos tipos importantes de puertos:

Puerto serial

  • Se utiliza para módems externos y ratones de ordenador más antiguos.

  • Dos versiones: 9 pin, modelo 25 pin.

  • Los datos viajan a 115 kilobits por segundo

Puerto paralelo

  • Utilizado para escáneres e impresoras.

  • También llamado puerto de impresora

  • Modelo de 25 pines

  • Puerto Centronics compatible con IEEE 1284

Puerto PS / 2

  • Usado para el teclado y el mouse de una computadora vieja

  • También llamado puerto de mouse

  • La mayoría de las computadoras antiguas ofrecen dos puertos PS / 2, cada uno para el mouse y el teclado

  • Puerto Centronics compatible con IEEE 1284

Puerto de bus serie universal (o USB)

  • Puede conectar todo tipo de dispositivos USB externos, como discos duros externos, impresoras, escáneres, mouse, teclados, etc.

  • Fue introducido en 1997.

  • La mayoría de las computadoras proporcionan dos puertos USB como mínimo.

  • Los datos viajan a 12 megabits por segundo.

  • Los dispositivos compatibles con USB pueden obtener energía de un puerto USB.

Puerto VGA

  • Conecta el monitor a la tarjeta de video de una computadora.

  • Tiene 15 hoyos.

  • Similar al conector del puerto serie. Sin embargo, el conector del puerto serie tiene patillas, el puerto VGA tiene orificios.

Conector de alimentación

  • Enchufe de tres puntas.

  • Se conecta al cable de alimentación de la computadora que se conecta a la barra de alimentación o al tomacorriente de pared.

Puerto firewire

  • Transfiere gran cantidad de datos a una velocidad muy rápida.

  • Conecta videocámaras y equipos de video a la computadora.

  • Los datos viajan de 400 a 800 megabits por segundo.

  • Inventado por Apple.

  • Tiene tres variantes: conector FireWire 400 de 4 pines, conector FireWire 400 de 6 pines y conector FireWire 800 de 9 pines.

Puerto de modem

  • Conecta el módem de una PC a la red telefónica.

Puerto Ethernet

  • Se conecta a una red e internet de alta velocidad.

(1 x 103)+ (2 x 102)+ (3 x 101)+ (4 x l00)

Computadora - Conversión de números

Hay muchos métodos o técnicas que pueden usarse para convertir números de una base a otra. En este capítulo, demostraremos lo siguiente:

  • Decimal a otro sistema base
  • Otro sistema base a decimal
  • Otro sistema base a no decimal
  • Método de acceso directo - Binario a octal
  • Método de acceso directo - octal a binario
  • Método de acceso directo - Binario a hexadecimal
  • Método de acceso directo: hexadecimal a binario

Decimal a otro sistema base

Paso 1 - Divide el número decimal a convertir por el valor de la nueva base.

Paso 2 : obtenga el resto del Paso 1 como el dígito más a la derecha (el dígito menos significativo) del nuevo número base.

Paso 3 - Divide el cociente de la división anterior por la nueva base.

Paso 4 : registre el resto del Paso 3 como el siguiente dígito (a la izquierda) del nuevo número base.

Repita los pasos 3 y 4, obteniendo los residuos de derecha a izquierda, hasta que el cociente se vuelva cero en el paso 3.

El último resto así obtenido será el dígito más significativo (MSD) del nuevo número base.

Ejemplo

Número decimal: 29 10

Cálculo Equivalente Binario -

Paso Operación Resultado Recordatorio
Paso 1 29/2 14 1
Paso 2 14/2 7 0
Paso 3 7/2 3 1
Etapa 4 3/2 1 1
Paso 5 1/2 0 1

Como se mencionó en los Pasos 2 y 4, los residuos deben organizarse en el orden inverso para que el primer resto se convierta en el dígito menos significativo (LSD) y el último resto se convierta en el dígito más significativo (MSD).

Número decimal: 29 10 = Número binario: 11101 2.

Otro sistema base al sistema decimal

Paso 1 - Determine el valor de la columna (posicional) de cada dígito (esto depende de la posición del dígito y la base del sistema numérico).

Paso 2 - Multiplique los valores de columna obtenidos (en el Paso 1) por los dígitos en las columnas correspondientes.

Paso 3 - Suma los productos calculados en el Paso 2. El total es el valor equivalente en decimal.

Ejemplo

Número Binario: 11101 2

Cálculo del equivalente decimal

Paso Número binario Número decimal
Paso 1 11101 2 ((1 x 2 4 ) + (1 x 2 3 ) + (1 x 2 2 ) + (0 x 2 1 ) + (1 x 2 0 )) 10
Paso 2 11101 2 (16 + 8 + 4 + 0 + 1) 10
Paso 3 11101 2 29 10

Número binario: 11101 2 = Número decimal: 29 10

Paso 1 - Convierta el número original a un número decimal (base 10).

Paso 2 - Convierta el número decimal así obtenido al nuevo número base.

Ejemplo

Número octal: 25 8

Paso 1 - Convertir a decimal

Paso Número octal Número decimal
Paso 1 25 8 ((2 x 8 1 ) + (5 x 8 0 )) 10

Etapa 4 2/2 1 0

Método de acceso directo ─ Binario a octal

Paso 1 - Divide los dígitos binarios en grupos de tres (comenzando desde la derecha).

Step 2 − Convert each group of three binary digits to one octal digit.

Example

Binary Number : 101012

Calculating Octal Equivalent −

Step Binary Number Octal Number
Step 1 101012 010 101
Step 2 101012 28 58
Step 3 101012 258

Binary Number : 101012 = Octal Number : 258

Shortcut Method ─ Octal to Binary

Step 1 − Convert each octal digit to a 3-digit binary number (the octal digits may be treated as decimal for this conversion).

Step 2 − Combine all the resulting binary groups (of 3 digits each) into a single binary number.

Example

Octal Number : 258

Calculating Binary Equivalent −

Step Octal Number Binary Number
Step 1 258 210 510
Step 2 258 0102 1012
Step 3 258 0101012

Octal Number : 258 = Binary Number : 101012

Shortcut Method ─ Binary to Hexadecimal

Step 1 − Divide the binary digits into groups of four (starting from the right).

Step 2 − Convert each group of four binary digits to one hexadecimal symbol.

Example

Binary Number : 101012

Step Binary Number Hexadecimal Number
Step 1 101012 0001 0101
Step 2 101012 110 510
Step 3 101012 1516

Step 1 − Convert each hexadecimal digit to a 4-digit binary number (the hexadecimal digits may be treated as decimal for this conversion).

Step 2 − Combine all the resulting binary groups (of 4 digits each) into a single binary number.

Computer - Data and Information

Data can be defined as a representation of facts, concepts, or instructions in a formalized manner, which should be suitable for communication, interpretation, or processing by human or electronic machine.

Data is represented with the help of characters such as alphabets (A-Z, a-z), digits (0-9) or special characters (+,-,/,*,<,>,= etc.)

What is Information?

Information is organized or classified data, which has some meaningful values for the receiver. Information is the processed data on which decisions and actions are based.

For the decision to be meaningful, the processed data must qualify for the following characteristics −

  • Timely − Information should be available when required.

  • Accuracy − Information should be accurate.

  • Completeness − Information should be complete.

Data Processing Cycle

Data processing is the re-structuring or re-ordering of data by people or machine to increase their usefulness and add values for a particular purpose. Data processing consists of the following basic steps - input, processing, and output. These three steps constitute the data processing cycle.

  • Input − In this step, the input data is prepared in some convenient form for processing. The form will depend on the processing machine. For example, when electronic computers are used, the input data can be recorded on any one of the several types of input medium, such as magnetic disks, tapes, and so on.

  • Processing − In this step, the input data is changed to produce data in a more useful form. For example, pay-checks can be calculated from the time cards, or a summary of sales for the month can be calculated from the sales orders.

  • Output − At this stage, the result of the proceeding processing step is collected. The particular form of the output data depends on the use of the data. For example, output data may be pay-checks for employees.

Computer - Networking

A computer network is a system in which multiple computers are connected to each other to share information and resources.

Characteristics of a Computer Network

  • Share resources from one computer to another.

  • Create files and store them in one computer, access those files from the other computer(s) connected over the network.

  • Connect a printer, scanner, or a fax machine to one computer within the network and let other computers of the network use the machines available over the network.

Following is the list of hardware's required to set up a computer network.

  • Network Cables
  • Distributors
  • Routers
  • Internal Network Cards
  • External Network Cards

Network Cables

Network cables are used to connect computers. The most commonly used cable is Category 5 cable RJ-45.

Distributors

A computer can be connected to another one via a serial port but if we need to connect many computers to produce a network, this serial connection will not work.

The solution is to use a central body to which other computers, printers, scanners, etc. can be connected and then this body will manage or distribute network traffic.

Router

A router is a type of device which acts as the central point among computers and other devices that are a part of the network. It is equipped with holes called ports. Computers and other devices are connected to a router using network cables. Now-a-days router comes in wireless modes using which computers can be connected without any physical cable.

Network Card

Network card is a necessary component of a computer without which a computer cannot be connected over a network. It is also known as the network adapter or Network Interface Card (NIC). Most branded computers have network card pre-installed. Network cards are of two types: Internal and External Network Cards.

Internal Network Cards

Motherboard has a slot for internal network card where it is to be inserted. Internal network cards are of two types in which the first type uses Peripheral Component Interconnect (PCI) connection, while the second type uses Industry Standard Architecture (ISA). Network cables are required to provide network access.

External Network Cards

External network cards are of two types: Wireless and USB based. Wireless network card needs to be inserted into the motherboard, however no network cable is required to connect to the network.

Universal Serial Bus (USB)

USB card is easy to use and connects via USB port. Computers automatically detect USB card and can install the drivers required to support the USB network card automatically.

Computer - Operating System

The Operating System is a program with the following features −

  • An operating system is a program that acts as an interface between the software and the computer hardware.

  • It is an integrated set of specialized programs used to manage overall resources and operations of the computer.

  • It is a specialized software that controls and monitors the execution of all other programs that reside in the computer, including application programs and other system software.

Objectives of Operating System

The objectives of the operating system are −

  • To make the computer system convenient to use in an efficient manner.

  • To hide the details of the hardware resources from the users.

  • To provide users a convenient interface to use the computer system.

  • To act as an intermediary between the hardware and its users, making it easier for the users to access and use other resources.

  • To manage the resources of a computer system.

  • To keep track of who is using which resource, granting resource requests, and mediating conflicting requests from different programs and users.

  • To provide efficient and fair sharing of resources among users and programs.

Characteristics of Operating System

Here is a list of some of the most prominent characteristic features of Operating Systems −

  • Memory Management − Keeps track of the primary memory, i.e. what part of it is in use by whom, what part is not in use, etc. and allocates the memory when a process or program requests it.

  • Processor Management − Allocates the processor (CPU) to a process and deallocates the processor when it is no longer required.

  • Device Management − Keeps track of all the devices. This is also called I/O controller that decides which process gets the device, when, and for how much time.

  • File Management − Allocates and de-allocates the resources and decides who gets the resources.

  • Security − Prevents unauthorized access to programs and data by means of passwords and other similar techniques.

  • Job Accounting − Keeps track of time and resources used by various jobs and/or users.

  • Control Over System Performance − Records delays between the request for a service and from the system.

  • Interaction with the Operators − Interaction may take place via the console of the computer in the form of instructions. The Operating System acknowledges the same, does the corresponding action, and informs the operation by a display screen.

  • Error-detecting Aids − Production of dumps, traces, error messages, and other debugging and error-detecting methods.

  • Coordination Between Other Software and Users − Coordination and assignment of compilers, interpreters, assemblers, and other software to the various users of the computer systems.

Computer - Internet and Intranet

In this chapter, we will see what is Internet and Intranet, as well as discuss the similarities and differences between the two.

Internet

It is a worldwide/global system of interconnected computer networks. It uses the standard Internet Protocol (TCP/IP). Every computer in Internet is identified by a unique IP address. IP Address is a unique set of numbers (such as 110.22.33.114) which identifies a computer’s location.

A special computer DNS (Domain Name Server) is used to provide a name to the IP Address so that the user can locate a computer by a name. For example, a DNS server will resolve a name https://www.tutorialspoint.com to a particular IP address to uniquely identify the computer on which this website is hosted.

Internet is accessible to every user all over the world.

Intranet

Intranet is the system in which multiple PCs are connected to each other. PCs in intranet are not available to the world outside the intranet. Usually each organization has its own Intranet network and members/employees of that organization can access the computers in their intranet.

Each computer in Intranet is also identified by an IP Address which is unique among the computers in that Intranet.

Similarities between Internet and Intranet

  • Intranet uses the internet protocols such as TCP/IP and FTP.

  • Intranet sites are accessible via the web browser in a similar way as websites in the internet. However, only members of Intranet network can access intranet hosted sites.

  • In Intranet, own instant messengers can be used as similar to yahoo messenger/gtalk over the internet.

Differences between Internet and Intranet

  • Internet is general to PCs all over the world whereas Intranet is specific to few PCs.

  • Internet provides a wider and better access to websites to a large population, whereas Intranet is restricted.

  • Internet is not as safe as Intranet. Intranet can be safely privatized as per the need.
