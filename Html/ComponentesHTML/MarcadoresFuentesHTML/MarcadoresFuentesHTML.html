Marcadores de fuentes HTML

B: Se utiliza para poner en negritas el texto

BIG: Se utiliza para convertir a una fuente más grande esto es interpretado por el navegador web

I: Se utiliza para convertir el texto en cursiva

S: Se utiliza para poner una raya en el texto

SMALL: Se utiliza para hacer el texto má s pequeño

STRIKE: Es el mismo resultado que s, pone una raya en el texto

TT: Se utiliza para dar formato de teletipo como se utiliza en los monitores e impresoras antiguos

U: Se utiliza para subrayar una palabra
